# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |
| < 0.1.0 | No |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Use GitHub Private Vulnerability Reporting in this repository:
- Open `Security` tab
- Click `Report a vulnerability`
- Share reproduction details, impact, and affected versions

If private reporting is unavailable, open a regular issue asking for a private contact channel and do not include exploit details.
